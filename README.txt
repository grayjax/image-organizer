Image Organizer
==============

This program helps organize your images into categories using AI.

How to Use:
1. Double-click ImageOrganizer.exe to start the program
2. Click "Change Folder" to select the folder containing your images
3. Click "Organize Images" to begin
4. The program will:
   - Show you each image as it processes
   - Create a new folder with today's date
   - Sort your images into categories
   - Keep your original images untouched

Features:
- Dark Mode toggle
- Preview window shows current image
- Progress bar shows completion status
- Cancel button to stop processing
- Settings button to customize categories

Need Help?
If you encounter any errors, check error_log.txt in the same folder.
